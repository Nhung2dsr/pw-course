## 2. JavaScript cơ bản
1. In kết quả ra màn hình
```JavaScript
console.log("Hello, wold!");
```
2. Khai báo biến
```JavaScript 
let tuoi = "22";
console.log("Tuổi: " + tuoi);
```
3. Hằng: **Là các giá trị không thể thay đổi được**
```JavaScript
const ten = "Phạm Nhung";
console.log(ten);
```
4. Boolean: **Kiểu dữ liệu logic chỉ trả về kết quả True/ False** 
```JavaScript
const dung = true;
const sai = false;
```
5. Câu điều kiện: **if...else,if...else if, switch case**
```JavaScript
const score = "5";
if (score < 5){
    console.log("Không đạt");
}
else (score >= 5){
    console.log("Đạt");
}
```
6. Vòng lặp
- _For_:
```JavaScript
let i = 0;
for (i = 0, i < 10, i++){
    console.log(i);
    i++;
}
```
- _While_:
```JavaScript
let i = 1;
while(i <= 5){
    console.log(i);
    i++;
}
```
